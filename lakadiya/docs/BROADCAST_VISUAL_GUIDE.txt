┌─────────────────────────────────────────────────────────────────────────────┐
│ ENDPOINTS SUMMARY                                                           │
└─────────────────────────────────────────────────────────────────────────────┘

POST /api/notifications/device-token
     → Store FCM token (called after login)
     → Target: Authenticated user only
     → Auto-called from mobile app

POST /api/notifications/send-test-otp
     → Send OTP to authenticated user
     → Target: One device (current user)
     → Use for testing OTP delivery

POST /api/notifications/broadcast-test
     → Send to ALL active devices
     → Target: Everyone logged in
     → Use for announcements

GET /api/notifications/logs?limit=20
     → View notification history
     → Target: Authenticated user's logs
     → Use for debugging


┌─────────────────────────────────────────────────────────────────────────────┐
│ DATABASE TABLES                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

device_tokens
├─ id: UUID (unique)
├─ user_id: UUID (foreign key)
├─ fcm_token: VARCHAR (500 chars, unique)
├─ device_type: VARCHAR (android/ios/web)
├─ is_active: BOOLEAN
├─ last_used: TIMESTAMPTZ
└─ created_at: TIMESTAMPTZ

notification_logs
├─ id: UUID
├─ user_id: UUID (nullable)
├─ fcm_token: VARCHAR
├─ title: VARCHAR
├─ body: TEXT
├─ data: JSONB (contains type, etc)
├─ status: VARCHAR (sent/failed/pending)
├─ error_msg: TEXT
└─ sent_at: TIMESTAMPTZ
